This file contains some notes how I installed certain prerequisites locally on my system.

It is outdated, does not match the exactly necessary dependencies, etc.

==============================================================


How to install the dependencies locally:

# Install PyQt5
...

export QT_SELECT=qt5.5.0-x86_64-linux-gnu

# Download & extract QScintilla
cd Qt4Qt5
qmake qscintilla.pro
make
make install


# Download & extract sip
python3 configure.py --sysroot /opt/sip 
# Edit siglib/Makefile: CPPFLAGS = -DNDEBUG -I. -I/opt/sip/include/python3.4m -I/usr/include/python3.4m 
make install


# Download & extract PyQt5
# Edit configure.py: Remove "if introspecting and target_config.qt_licensee != 'Builder Qt' and ltype == 'GPL': [...]"
sudo apt-get remove sip-dev  # To avoid using old .h files
export CPLUS_INCLUDE_PATH=/opt/sip/include/python3.4m/:/usr/include/python3.4m/
export C_INCLUDE_PATH="$CPLUS_INCLUDE_PATH"
python3 configure.py --sip /opt/sip/bin/sip --qsci-api -v /opt/sip/sip/PyQt5 --sysroot /opt/pyqt5
make install


# Install Qscintilla Python bindings: (in QSci installer dir)
python3 configure.py --pyqt PyQt5 -d /opt/pyqt5/lib/python3/dist-packages/PyQt5/ --sip /opt/sip/bin/sip -w --pyqt-sipdir /opt/sip/sip/PyQt5


# Download & extract Eric6 IDE
export PYTHONPATH
PYTHONPATH+=:/opt/pyqt5/lib/python3/dist-packages
PYTHONPATH+=:/opt/sip/lib/python3/dist-packages
python3 install.py --pyqt=5 -b /opt/eric/bin -d /opt/eric/python3




# Installing cx-freeze on Ubuntu
pip3 install --no-install cx-freeze
cd /tmp/pip_build_unruh/cx-freeze/
Change: if not vars.get("Py_ENABLE_SHARED", 0): --> If True:
sudo pip3 install --no-clean . 



# EclipseFP on Ubuntu
sudo apt-get install ghc cabal-install happy
cabal update
cabal install buildwrapper # needed?
cabal install hoogle hlint stylish-haskell # needed?
cabal install cabal-install
Install eclipse
Install EclipseFP via its Eclipse update site
When EclipseFP wants to build scion-browser, cancel
In Preferences->Haskell->Helper Executables, choose cabal 1.22
Create and make writable: /opt/eclipse/plugins/.eclipsefp/
Restart eclipse
When EclipseFP wants to build scion-browser, accept (but deselect building of helpers)




====================================== On windows:

Install Python3 (32bit!)
Install PyQt5 (32bit)  -- 64 didn't work (even with 64 python)

pip3 install mypy-lang

From http://www.lfd.uci.edu/~gohlke/pythonlibs/#lxml, download lxml‑3.4.4‑cp34‑none‑win32.whl
pip3 install lxml‑3.4.4‑cp34‑none‑win32.whl

Install OCaml with Cygwin: 
http://protz.github.io/ocaml-installer/

Install Batteries from Source:
http://forge.ocamlcore.org/frs/?group_id=17
make
make install

Install xml-light:
http://tech.motion-twin.com/xmllight.html
make
make INSTALLDIR=/cygdrive/c/OCaml/lib install

Install haskell:
From https://www.haskell.org/downloads/windows, MinGHC installer for GHC 7.10.2 (32-bit)
cabal update
In math-ed\haskell:
cabal install --only-dependencies
cabal build


To just start: in math-ed (with cygwin):
make cmathml.exe
python3 main.py

Then to generate the distribution:
make
